# pAInt Local - Personal AI Neural Toolkit

**Свободная версия AmuseAI на Python без цензуры.**

## Возможности
- 🖼️ **Генерация изображений**: Text-to-Image, Image-to-Image, Inpainting (подготовка)
- 📝 **Текстовые модели**: Поддержка LLM (Llama, Mistral и др.)
- 🎥 **Видео**: Поддержка видео-моделей (структура готова)
- ⚙️ **Гибридный режим**: Ползунок распределения нагрузки CPU ↔ GPU
- 📚 **Библиотека моделей**: Авто-сканирование папок, поддержка мульти-файловых структур
- 🔓 **Без цензуры**: Полный контроль над контентом

## Структура папок
```
pAInt-local/
├── src/
│   └── main.py          # Основной код приложения
├── models/              # Папка для ваших моделей
│   ├── text/            # LLM модели (.gguf, .bin)
│   ├── image/           # Diffusion модели (.safetensors, .ckpt)
│   ├── video/           # Видео модели
│   ├── lora/            # LoRA адаптеры
│   └── custom/          # Пользовательские модели
├── requirements.txt     # Зависимости
└── README.md
```

## Установка

1. **Установите зависимости:**
   ```bash
   pip install -r requirements.txt
   ```

2. **Запустите приложение:**
   ```bash
   python src/main.py
   ```

3. **Добавьте модели:**
   - Скачайте модели (например, с Civitai или HuggingFace).
   - Положите файлы в соответствующую папку `models/image/` или `models/text/`.
   - Или используйте кнопку "+ Добавить модель" в интерфейсе.

## Как это работает
- **GUI**: Построен на `customtkinter` (современный темный интерфейс).
- **Движок**: В текущей версии реализована архитектура и интерфейс. Для реальной генерации необходимо раскомментировать и настроить вызовы `diffusers` (для изображений) и `llama-cpp-python` или `transformers` (для текста) в классе `AIEngine`.
- **Производительность**: Параметр `gpu_ratio` в коде отвечает за распределение слоев модели между RAM и VRAM (требуется доработка под конкретные библиотеки).
